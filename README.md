🛍️ Customer Segmentation using K-Means Clustering
📌 Project Overview

This project uses Machine Learning (K-Means Clustering) to segment customers based on their purchasing behavior.
It helps businesses identify different customer groups and design targeted marketing strategies.

An interactive Streamlit web app is also built to allow users to input customer details and get real-time segmentation results.

🎯 Objective
Group customers based on income and spending behavior
Identify high-value and low-engagement customers
Support data-driven marketing decisions
📊 Dataset
Dataset: Mall Customers Dataset
Features:
Age
Gender
Annual Income (k$)
Spending Score (1–100)
Records: ~200 customers
⚙️ Tech Stack
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Streamlit
🤖 Machine Learning Approach
Algorithm: K-Means Clustering (Unsupervised Learning)
Feature Selection:
Annual Income
Spending Score
Data Scaling using StandardScaler
Optimal clusters determined using Elbow Method
📈 Results & Insights

Customers were segmented into 5 groups:

💎 High Income, High Spending → Premium Customers
💡 High Income, Low Spending → Potential Customers
🛒 Low Income, High Spending → Impulsive Buyers
📉 Low Income, Low Spending → Low Value Customers
📊 Average Customers
